## :computer:How to run on your machine



```
- Clone repository

$ git clone https://github.com/allanrodriguest/mastering-pomodoro.git

```

```
- At the root of the project, run:

yarn or npm install
```

```
- To run the app in your machine:

yarn dev or npm run dev
```
